## Clinic Operations & KPI Analysis

This project demonstrates how hospital operational data can be used to identify patient flow bottlenecks, scheduling inefficiencies, and process gaps that impact waiting time, doctor utilization, and overall patient experience.

- Case Study Document: Clinic_Operations_Case_Study.pdf  
- Dataset (Google Sheet):

### Key Focus Areas

- Patient waiting time analysis  
- Peak vs off-peak congestion  
- Doctor workload distribution  
- Department-level performance  
- Data quality gaps  

### Purpose

This project showcases how data can support hospital management in improving efficiency, patient experience, and operational visibility.
